# Exercício 19 — Rate limiting no SSH (anti força bruta)

## UFW
```bash
sudo ufw limit 22/tcp
```

## iptables
Exemplo usando o módulo `recent` (bloqueia se passar de 3 novas conexões em 60s):

```bash
# 1) Se exceder, derruba
sudo iptables -A INPUT -p tcp --dport 22 -m conntrack --ctstate NEW \
  -m recent --update --seconds 60 --hitcount 4 --name SSH -j DROP

# 2) Marca tentativas novas
sudo iptables -A INPUT -p tcp --dport 22 -m conntrack --ctstate NEW \
  -m recent --set --name SSH -j ACCEPT
```
