# Exercício 21 — NAT: redirecionar 80 → 8080 local (port forward)

## UFW
O UFW não tem um comando único para DNAT/REDIRECT; o padrão é configurar no arquivo e recarregar.

```bash
# Permitir a porta do app (porque após o redirect o destino vira 8080)
sudo ufw allow 8080/tcp

# Editar as regras NAT do UFW
sudo nano /etc/ufw/before.rules

# Recarregar
sudo ufw reload
```

Regra típica a adicionar na seção `*nat` (antes do `COMMIT`):

```text
-A PREROUTING -p tcp --dport 80 -j REDIRECT --to-ports 8080
```

## iptables
```bash
# Redireciona conexões que chegam na porta 80 para 8080 local
sudo iptables -t nat -A PREROUTING -p tcp --dport 80 -j REDIRECT --to-ports 8080

# Libera a porta 8080 na tabela filter (porque ela será o destino após o redirect)
sudo iptables -A INPUT -p tcp --dport 8080 -j ACCEPT
```
