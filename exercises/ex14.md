# Exercício 14 — Liberar MySQL (3306/tcp) apenas pela interface eth1

## UFW
```bash
sudo ufw allow in on eth1 to any port 3306 proto tcp
```

## iptables
```bash
sudo iptables -A INPUT -i eth1 -p tcp --dport 3306 -j ACCEPT
```

Observação: normalmente isso faz sentido com política padrão `DROP/DENY` na entrada.
