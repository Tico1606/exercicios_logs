# Exercício 09 — Regras essenciais (loopback e conexões estabelecidas)

## UFW
```bash
# Loopback (recomendado)
sudo ufw allow in on lo
sudo ufw allow out on lo
```

Para conexões já estabelecidas (ESTABLISHED,RELATED), o UFW normalmente já permite isso nas regras internas (before.rules), então não existe um comando “ufw allow established” via CLI.

## iptables
```bash
# Permitir loopback
sudo iptables -A INPUT -i lo -j ACCEPT
sudo iptables -A OUTPUT -o lo -j ACCEPT

# Permitir conexões já estabelecidas/relacionadas
sudo iptables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
```
