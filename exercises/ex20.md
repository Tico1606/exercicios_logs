# Exercício 20 — Logging: registrar tentativas na 5432 antes de bloquear

## UFW
```bash
# Regra que loga e bloqueia
sudo ufw deny log 5432/tcp
```

## iptables
```bash
# 1) Loga
sudo iptables -A INPUT -p tcp --dport 5432 -j LOG --log-prefix "DROP 5432 " --log-level 4

# 2) Bloqueia
sudo iptables -A INPUT -p tcp --dport 5432 -j DROP
```
