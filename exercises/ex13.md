# Exercício 13 — Liberar SSH (22/tcp) só para 192.168.1.100

## UFW
```bash
sudo ufw allow from 192.168.1.100 to any port 22 proto tcp
```

## iptables
```bash
sudo iptables -A INPUT -p tcp -s 192.168.1.100 --dport 22 -j ACCEPT
```

Observação: para ficar “apenas esse IP”, você normalmente combina com política padrão `DROP/DENY` para entrada.
