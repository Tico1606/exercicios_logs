# Exercício 02 — Habilitar e desabilitar

## UFW
```bash
sudo ufw enable
sudo ufw disable
```

## iptables
O iptables não tem um “enable/disable” simples porque ele é apenas um utilitário para **programar regras no kernel (netfilter)**; as regras ficam ativas assim que você as aplica.

```bash
# “Desabilitar na prática”: limpar regras e voltar políticas para ACCEPT
sudo iptables -P INPUT ACCEPT
sudo iptables -P FORWARD ACCEPT
sudo iptables -P OUTPUT ACCEPT
sudo iptables -F
sudo iptables -t nat -F
sudo iptables -X

# “Habilitar na prática”: restaurar um conjunto salvo (exemplo)
# sudo iptables-save | sudo tee /etc/iptables/rules.v4
# sudo iptables-restore < /etc/iptables/rules.v4
```
