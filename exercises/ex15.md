# Exercício 15 — Bloquear pacotes INVALID (conntrack)

## UFW
O UFW não tem um comando direto de CLI para “ctstate INVALID”. O jeito comum é adicionar a regra nas regras internas e recarregar.

```bash
# Editar e adicionar a regra (exemplo de comando para abrir editor)
sudo nano /etc/ufw/before.rules

# Depois recarregar o UFW
sudo ufw reload
```

Linha típica a adicionar na chain `ufw-before-input`:

```text
-A ufw-before-input -m conntrack --ctstate INVALID -j DROP
```

## iptables
```bash
sudo iptables -A INPUT -m conntrack --ctstate INVALID -j DROP
```
